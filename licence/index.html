This website relies on a few interesting bits to get it to work.

Hexo

With the exception of index.html, this site is built with Hexo — a Node.js static site generator.

This site utilises a modified version of the Apollo theme for Hexo, created by pinggod. It is MIT-licenced.

Open Source

I learned a great deal making this site, and I believe others should benefit from such an experience, therefore the following directories (and their publicly-accessible links) are open-source:

  • rory.cc/index.html
  • rory.cc/blog/
  • rory.cc/code/
  • rory.cc/music/
